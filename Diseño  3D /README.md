# Diseño3D 

## Cual es el objetivo

- El proyecto trata sobre diseñar en 3D un nuevo diseño para el invernadero como distribución de pilares y suelo  cuales se dividieron en partes para cada integrante de clase teniendo pilares y suelos distintos.

  # Pilares

 - El  inicio del  proyecto fue con los pilares cuales se mando hacer autonoma por la falta de asistencia del profesorado los pilares pueden tener o una "salida" o una "entrada" o varias normalmente 2 o hasta 4. Para que se unene los pilares para que así los pilares estén unidos y estables y no se caigan ya que se conectan unos con otros formando como un caparón, se mandaron a hacer 2 por cada persona

   
 ![pilares](imagenes/pilartry.png)

- Los pilares tienen unnas medidas exacta de largo y ancho si tiene salidas igual que tienen una medidas exactas las entrandas añadiendo centimetros

  # Suelo

- El suelo es muy parecido a los pilares también siendo  distribuido entre alumnos y con entradas y salidas solo que con distintas medidas y en si funcionnes el suelo esta enumerados por letras y números

 ![suelo](imagenes/3D_qui.png)

- Es suelo debe de tener un espacio para que los pilares encagen y no se descoloquen permitiendo una mayor estabilidad y tambien espacios para que otros suelos encagen en el haciendo que sea menos probable el desoclocamiento.

- Cuando el diseño  3D está preparado y se imprima recordando que número tiene asignado se conecta con los pilares y otros suelos

  # Final del proyecto

  - Al juntar las piezas y los suelos formando la estrucutra del diseño 3D del invernadero debe de quedar una estrucutra perfectamente concluyendo como un rectanguló y así una estructura totalmente estable para el  invernadero.
